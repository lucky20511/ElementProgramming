### Collections Libraries

```
Collections.binarySearch(C, 42) // Collections.binarySearch(C, 'f')
Collections.sort(C) // Collections.sort(C, cmp)
Collections.min(C) // Collections.min(C, cmp)
Collections.max(C) // Collections.max(C, cmp)
Collections.fill(C, 5) // Collections.fill(C, 'f')
Collections.reverse(C)
Collections.rotate(C, 5)
Collections.swap(C, 0, 2)
```
