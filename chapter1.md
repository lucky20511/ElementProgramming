# Strategies for Interview

#### Resume

1.Clear statement of Objective

2.The ones differentiate you from others

3.High quality with no errors

#### Coding for Interview v.s Coding in industry

1.Take correctness, simplicity, space and time complexity as top priority.

2.Keep variable name short and clear.

3.Don't need to use setter and getter

Approaching the problem

1.Clarify the problem

2.Work on concrete examples

3.Spell out brute-force solution

4.Think out loud

5.Apply patterns
